    CASE FOR IF/ELSE WHILE VETORES MATRIZES

CASE

O "CASE" tem como função determinar diferesntes casos (como o próprio nome em inglês) para aqueles programas em específico, onde a pessoa selecionará uma opção e terá um determinado caso para aqula escolha.

FOR

O "FOR" é usado para executar um conjunto de instroções de cordo com uma quantidade exigida.

IF/ELSE

O "IF" e "ELSE" são um conjunto de instruções que andam junto, porem, para trabalharem um contra o outro, já que o "If" exibirá o programa caso a afirmação seja correta, caso não seja, o "ELSE" será executado.

WHILE

A estrutura de repetição "while" é um recurso para organizar melhor e realizar testes em blocos de código. A repetição while é ideal para realizar testes em que não se conhece o início e o fim da condição, se tornando uma ferramenta muito importante.

VETORES

Vetores são estruturas de dados que armazenam usualmente uma quantidade fixa de dados de um certo tipo; por esta razão, também são conhecidos como estruturas homogêneas de dados.

MATRIZES

Uma matriz é um vetor de vetores, todos de mesmo tamanho. Em Java, é possível criar um vetor de vetores onde cada elemento tem um tamanho diferente.
